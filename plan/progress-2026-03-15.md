## Changes

### Display fixes
- Reverted time format from HH:MM:SS to HH:MM, updating only when second == 0 (eliminates flicker)
- Moved time textbox from x=160 to x=200 to avoid overwriting long weekday names

### Alert auto-dismiss after 10 seconds
- FocusReminder gains `alert_duration` parameter (default 20 ticks = 10s at 0.5s interval)
- Alert auto-dismisses when tick counter exceeds duration
- Auto-dismissed alert lets subsequent alerts fire normally

### Weekday display updates overnight
- Bug: weekday was set once in `start()` and never updated

- Renamed `status()` to `weekday()` to eliminate repeated `_format_weekday()` calls

- Tests now take a single fixture parameter instead of two
