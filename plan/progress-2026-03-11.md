# Progress Report — 2026-03-11

## Summary

Fixed display flickering and overlap, and added auto-dismiss for alerts after 10 seconds.

## Changes

### Time display reverted to HH:MM, updates once per minute
- `_format_current_time()` returns `HH:MM` instead of `HH:MM:SS`
- `tick()` only updates the time display when `second == 0`
- Eliminates flickering caused by clearing and redrawing every 0.5s tick

### Time textbox moved right to avoid overlap
- `_time_display` x position moved from 160 to 200
- Width reduced from 140 to 100 (HH:MM is shorter than HH:MM:SS)
- Prevents overwriting long weekday names like "Wednesday"

### Alert auto-dismisses after 10 seconds
- `FocusReminder` gains `alert_duration` parameter (default 20 ticks = 10s)
- Tick counter increments while an alert is active
- Alert is auto-dismissed when counter exceeds duration
- Counter resets on dismiss (button, auto, or no active alert)
- Auto-dismissed alert lets next alert fire normally
